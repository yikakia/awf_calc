# 终落之城计算器

用于量化[终落之城](https://store.steampowered.com/app/2706020/ALL_WILL_FALL_PhysicsBased_Survival_City_Builder/)中的产出

## 功能
- [ ] 可选首领技能计算
  - [ ] 工人
  - [ ] 水手 (目前计算强制开启技能)
  - [ ] 工程师 (目前计算强制开启技能)
- [ ] 可选首领效率计算(5%)
- [ ] 同建筑物分配多种居民
- [ ] 建筑物属性配置化
- [x] 翻译
  - [x] 中文
  - [x] English

## 开发
项目以 Golang 为开发语言，采用 [fyne.io](https://docs.fyne.io/) 作为GUI框架
